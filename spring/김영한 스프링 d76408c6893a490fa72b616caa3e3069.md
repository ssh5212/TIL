단축키

alt + shift + v = 앞 부분 자동 완성

ctrl + shift + enter = 뒷 부분 자동 완성

ctrl + shift + t = 테스트 만들기

ctrl + alt + b / ctrl + alt + click = implementation으로 바로 이동

ctrl + r = rename

ctrl + t = refactor (수정)

ctrl + shift + t = 테스트 만들기

ctrl + e = recent files 최근 사용한 파일 목록 (수정)

ctrl + f5 = 최근 실행 다시 실행

ctrl + f6 = 현재 위치 실행 (수정)
